# chandraukom
-----------------
Sinopsis Aplikasi
------------------
Aplikasi ini adalah aplikasi yang dikhususkan untuk membantu orang atau user yang ingin mencari link atau tutorial - tutorial namun belum mengetahui alamat url yang benar. Di aplikasi ini menyajikan tutorial – tutorial yang sudah di kelompokka berdasarkan kategorinya guna untuk memudahkan user untuk mencari link atau tutorial - tutorial.

Manfaat
-----------------
- Memudahkan user dalam mencari alamat url atau tutorial – tutorial.
- User Bisa Melihat Tutorial Lansung di Aplikasi

Pengguna
-----------------
1. Anak - anak
2. Dewasa / Remaja
3. Orang Tua

Kegiatan
-----------------
1. User Bisa Melihat Link - atau Tutorial
2. User Bisa Menginput Link atau Berbagi Link
3. User Bisa Melihat Langsung Tutorial Melalui Aplikasi

Fitur
-----------------
1. Menginput Data
2. Melihat Data
3. Mempunyai Web Browser Sendiri
